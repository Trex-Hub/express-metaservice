# Express JS 

This project is a template for express js.

## Prerequisites

- Node.js
- pnpm
- Inngest CLI

## Installation

Install the project dependencies:

```bash
pnpm install
```

## Development

1. Start the development server:

```bash
pnpm dev
```

## Project Structure

The project uses:
- TypeScript for type safety

## Environment Variables

Make sure to set up your environment variables in a `.env` file. Check `.env.example` for required variables.
